# MedOCR 🩺

A versatile OCR toolkit designed for medical documents—such as prescriptions, lab reports, and patient records—leveraging both **Tesseract** and **Gemma** (Google’s latest multimodal model) for superior accuracy and flexibility.

---

## 🚀 Key Features

- 🔧 **Preprocessing**: Deskew, denoise, binarize, resize for optimal OCR performance.
- 🧠 **Post-processing**: Medical spell-check, terminology normalization, structured JSON/CSV output.
- 🛠️ **Modular architecture**: Swap pipelines, add custom models, extend processing.
- 💻 **CLI & Python API**: Integrate into scripts or workflows seamlessly.

---

## 📦 Installation

Clone and install dependencies:

```bash
git clone https://github.com/DeepLumiere/MedOCR.git
cd MedOCR
pip install -r requirements.txt
```
